---
title: "Token Authentication Enabled Jupter Server"
---

When you see a "Token Authentication Enabled" when trying to log into Jupyter server here are the steps to take to login.

<Frame caption="Jupyter Server Token Authentication Screen">
  <img src="/images/jupyter-server-token.png" alt="Jupyter Server Token Authentication" />
</Frame>

1. Go to your Pod and click `connect` button
2. There should be a `Web Terminal` start button
3. Click start and then open the web terminal
4. In the terminal run the command: `jupyter server list`

You will get a response such as:
```bash
root@2779b5db68b8:/# jupyter server list
Currently running servers:
http://localhost:8888/?token=ua5nw5fwkdzseqpp5apj :: /
root@2779b5db68b8:/#
```

The token are the digits following the `=` sign: `ua5nw5fwkdzseqpp5apj`. You can try to now go back to jupyter notebook and in the top "password or token" input box input the token to login.
