View your account information and current balance.

<RequestExample>
```bash Command
runpodctl user
```
</RequestExample>

## Alias

You can use `me` as a shorthand for `user`:

```bash
runpodctl me
```

## Description

The `user` command displays information about your Runpod account, including:

- Account email
- Current credit balance
- Account status

## Example

```bash
runpodctl user
```
