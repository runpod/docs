Create new Runpod resources, including individual Pods or groups of Pods with identical configurations.

## Available commands

- [`runpodctl create pod`](/runpodctl/reference/runpodctl-create-pod) - Create a single Pod
- [`runpodctl create pods`](/runpodctl/reference/runpodctl-create-pods) - Create multiple Pods with identical configurations
