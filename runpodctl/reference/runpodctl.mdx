The official Runpod command-line interface for managing Pods, transferring files, and interacting with your Runpod resources from the terminal.

All Pods come with `runpodctl` pre-installed with a Pod-scoped API key, making it easy to manage your Pods directly from the command line.

## Installation

Choose one of the following methods to install `runpodctl` on your local machine.

### macOS

**Homebrew**

```sh
brew install runpod/runpodctl/runpodctl
```

**ARM (Apple Silicon)**

```sh
wget --quiet --show-progress https://github.com/runpod/runpodctl/releases/download/v1.14.3/runpodctl-darwin-arm64 -O runpodctl && chmod +x runpodctl && sudo mv runpodctl /usr/local/bin/runpodctl
```

**AMD (Intel)**

```sh
wget --quiet --show-progress https://github.com/runpod/runpodctl/releases/download/v1.14.3/runpodctl-darwin-amd64 -O runpodctl && chmod +x runpodctl && sudo mv runpodctl /usr/local/bin/runpodctl
```

### Linux

```sh
wget --quiet --show-progress https://github.com/Run-Pod/runpodctl/releases/download/v1.14.3/runpodctl-linux-amd64 -O runpodctl && chmod +x runpodctl && sudo cp runpodctl /usr/bin/runpodctl
```

### Windows (PowerShell)

```sh
wget https://github.com/runpod/runpodctl/releases/download/v1.14.3/runpodctl-win-amd -O runpodctl.exe
```

## Available commands

### Resource management

- [`runpodctl get`](/runpodctl/reference/runpodctl-get) - Retrieve information about Pods and cloud GPUs
- [`runpodctl create`](/runpodctl/reference/runpodctl-create) - Create new Pods
- [`runpodctl start`](/runpodctl/reference/runpodctl-start) - Start stopped Pods
- [`runpodctl stop`](/runpodctl/reference/runpodctl-stop) - Stop running Pods
- [`runpodctl remove`](/runpodctl/reference/runpodctl-remove) - Remove Pods permanently

### File transfer

- [`runpodctl send`](/runpodctl/reference/runpodctl-send) - Send files or folders to a Pod or another machine
- [`runpodctl receive`](/runpodctl/reference/runpodctl-receive) - Receive files or folders from another machine

### SSH management

- [`runpodctl ssh`](/runpodctl/reference/runpodctl-ssh) - Manage SSH keys and connections

### Configuration and utilities

- [`runpodctl config`](/runpodctl/reference/runpodctl-config) - Configure CLI settings and API credentials
- [`runpodctl update`](/runpodctl/reference/runpodctl-update) - Update to the latest version
- [`runpodctl version`](/runpodctl/reference/runpodctl-version) - Display the current version

## Getting started

After installation, configure `runpodctl` with your API key:

```sh
runpodctl config --apiKey YOUR_API_KEY
```

You can generate an API key from the [Runpod console](https://www.runpod.io/console/user/settings).
