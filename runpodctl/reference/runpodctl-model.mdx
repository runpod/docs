List and manage models in your Runpod account.

<RequestExample>
```bash Command
runpodctl model <subcommand> [flags]
```
</RequestExample>

## Subcommands

### List models

List your models:

```bash
# List your models
runpodctl model list

# List all available models
runpodctl model list --all

# Filter by name
runpodctl model list --name "llama"

# Filter by provider
runpodctl model list --provider "meta"
```

#### List flags

<ResponseField name="--all" type="bool">
List all available models, not just your own.
</ResponseField>

<ResponseField name="--name" type="string">
Filter by model name.
</ResponseField>

<ResponseField name="--provider" type="string">
Filter by model provider.
</ResponseField>

### Add a model

Add a model to your account:

```bash
runpodctl model add --name "my-model" --model-path ./model
```

#### Add flags

<ResponseField name="--name" type="string">
Model name.
</ResponseField>

<ResponseField name="--model-path" type="string">
Directory containing model files to upload.
</ResponseField>

<ResponseField name="--create-upload" type="bool">
Create an upload session.
</ResponseField>

<ResponseField name="--file-name" type="string">
File name for upload.
</ResponseField>

<ResponseField name="--file-size" type="string">
File size in bytes.
</ResponseField>

<ResponseField name="--content-type" type="string">
Upload content type.
</ResponseField>

<ResponseField name="--part-size" type="string">
Multipart upload part size in bytes.
</ResponseField>

<ResponseField name="--metadata" type="string">
Metadata key=value pairs.
</ResponseField>

<ResponseField name="--model-status" type="string">
Initial model status.
</ResponseField>

<ResponseField name="--version-status" type="string">
Initial model version status.
</ResponseField>

<ResponseField name="--credential-type" type="string">
Credential type (if required).
</ResponseField>

<ResponseField name="--credential-reference" type="string">
Credential reference (if required).
</ResponseField>

### Remove a model

Remove a model from your account:

```bash
runpodctl model remove --name "my-model"
```

#### Remove flags

<ResponseField name="--name" type="string">
Model name.
</ResponseField>

<ResponseField name="--owner" type="string">
Model owner.
</ResponseField>
