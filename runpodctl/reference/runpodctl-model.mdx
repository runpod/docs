List and manage models in your Runpod account.

<RequestExample>
```bash Command
runpodctl model <subcommand> [flags]
```
</RequestExample>

## Subcommands

### List models

List your models:

```bash
# List your models
runpodctl model list

# List all available models
runpodctl model list --all

# Filter by name
runpodctl model list --name "llama"

# Filter by provider
runpodctl model list --provider "meta"
```

#### List flags

| Flag | Type | Description |
|------|------|-------------|
| `--all` | bool | List all available models |
| `--name` | string | Filter by model name |
| `--provider` | string | Filter by model provider |

### Add a model

Add a model to your account:

```bash
runpodctl model add --name "my-model" --model-path ./model
```

#### Add flags

| Flag | Type | Description |
|------|------|-------------|
| `--name` | string | Model name (required) |
| `--model-path` | string | Path to the model files (required) |

### Remove a model

Remove a model from your account:

```bash
runpodctl model remove --name "my-model"
```
